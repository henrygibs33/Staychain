# 🏠 Staychain - Decentralized Vacation Rentals

> 🚀 A peer-to-peer vacation rental platform built on Stacks blockchain, enabling trustless bookings without intermediaries!

## 🌟 Overview

Staychain is a decentralized Airbnb-style booking system that allows property owners to list their vacation rentals and guests to book them directly through smart contracts. No middleman, no excessive fees - just pure P2P magic! ✨

## 🎯 Key Features

- 🏡 **Property Listings** - Create and manage vacation rental properties
- 📅 **Smart Bookings** - Automated booking system with date availability
- 💰 **Escrow Payments** - Secure STX payments held in smart contract
- ⭐ **Review System** - Rate and review properties after completed stays
- 👤 **User Profiles** - Track hosting and booking history
- 🔒 **Trustless** - No central authority required

## 🛠️ Core Functions

### For Property Owners (Hosts)

- `create-property` - List a new vacation rental
- `update-property` - Modify property details
- `toggle-property-status` - Activate/deactivate listings
- `complete-booking` - Release escrowed funds after guest checkout

### For Guests

- `book-property` - Reserve a property for specific dates
- `cancel-booking` - Cancel reservation and get refund
- `add-review` - Rate and review completed stays
